

Muistilista GitHubin käytöstä

- Tee uusi tiedosto
	- touch filename.filetype

- Avaa tiedosto tekstieditorissa
	- code filename.filetype (avaa Visual Studio Codessa)

- GitHubin status
	- git status
	- punainen tarkoittaa, että tiedoston commit puuttuu

- Tiedoston lisääminen
	- git add filename.filetype
	- punainen väri pitäisi vaihtua vihreään

- Commit
	- git commit -m "Kuvaus tiedostosta" filename.filetype

- Push 
	- git push
	- Anna kirjautumistiedot

- Doned
	- Good Job






Lisää komentoja

- git branch
	- näyttää missä branch:issä olet tällä hetkellä (*) ja myös muiden branch:ien nimet

- git checkout branchname
	- vaihtaa branchin
